Assumptions

  1. X is horizontal
  2. Y is vertical
  3. Dimensions are in metres, ie 3,4 is 3m wide by 4m high.
  4. That directions map on to a clock face with
    • North (0) on 12
    • East (90) on 3
    • South (180) on 6
    • West (270) on 9
  5. A command that positions the drone on the boundary is not that same as one that attempts to breach the boundary.
  6. Toggling the lights will change lights that are 'off' to 'on', and lights that are 'on' to 'off'.
  7. Flashing the lights will leave them in an 'off' state.

Observations

  • Sin(x) = Opposite / Hypotenuse
  • Cos(x) = Adjacent / Hypotenuse
  • Tan(x) = Opposite / Adjacent
  • Opposite = Sin(x) * Hypotenuse = Tan(x) * Adjacent
  • Adjacent = Cos(x) * Hypotenuse = Opposite / Tan(x)
  • Hypotenuse = Opposite / Sin(x) = Adjacent / Cos(x)
  • Opposite = X
  • Adjacent = Y
  • Hypotenuse = Distance
  • Radians = Degrees * pi / 180
  • Degrees = Radians * 180 / pi
  • Ready reckoner...
    • North -> 360 Degrees = 2pi Radians. (0 Degrees = 0pi Radians.)
    • South -> 180 Degrees = pi Radians.
    • East -> 90 Degrees = pi/2 Radians.
    • West -> 270 Degrees = 3pi/2 Radians.
