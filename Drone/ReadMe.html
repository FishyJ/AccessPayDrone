Assumptions

  1. X is horizontal
  2. Y is vertical
  3. Dimensions are in metres, ie 3,4 is 3m wide by 4m high.
  4. That directions map on to a clock face with
    • North (0) on 12
    • East (90) on 3
    • South (180) on 6
    • West (270) on 9

Observations

  • Sin(a) = Opposite / Hypotenuse
  • Cos(a) = Adjacent / Hypotenuse
  • Opposite = Sin(a) * Hypotenuse
  • Adjacent = Cos(a) * Hypotenuse
  • Hypotenuse = Opposite / Sin(a) = Adjacent / Cos(a)
  • Opposite = X
  • Adjacent = Y
  • Hypotenuse = Distance
