APD001-XE02 Specification

AccessPay has an office Drone.

It has a number of features and capabilities that can be controlled via a command interface.

Management commands:

Start (S) start the drone
Boundary (Bx,y) define the drones navigable, rectangular border where x,y 
are the maximum upper right coordinates; the lower right is assumed to be 
(0,0). 
InitialPosition (Px,y) define the initial position of the drone; (0,0) is default
Restart (R) restart the drone, losing Boundary and InitialPosition settings
Shutdown (D) shutdown the drone

The system must be started (Start) and must also have received the Boundary command before any Action commands can be processed.

Action commands:

Toggle lights (T) turn the light on
Flash lights (F) flash the lights
Alert (horn) (At) sound the horn for t seconds
Home (H) navigate home
Move (Mt,d) move for t seconds time in d direction (compass heading)

The drone has a single speed engine of 0.5 m/s, reaches top speed in 0 milliseconds (impressive, and highly confidential technology), operates at a fixed elevation of 2.1m and can head in numerous directions depending on the navigation module installed. The drone can accept a stream of Commands, but only processes them sequentially.

There are two navigation modules (nav-mods) our Drone supports; Rails and Freedom. Rails only supports Move commands with a direction value of 0, 90, 180 and 270. Freedom is much more advanced and supports full 360-degree movement across the plane. The Drone tracks its current position as an important input into an on-board critical safety feature.

Whilst our Drone will accept any Move instruction it will never go beyond the configured Boundary. If it receives an instruction that results in it crossing the boundary it will go as far as it can and then sound the horn 3 times. Boundary detection relies on the Drone tracking its movement and knowing its current coordinates in relation to the boundaries.

Implement the Drone software logic, providing a test suite, that uses the Command Interface and asserts using the state of the Drone.
